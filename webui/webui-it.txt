WEBUI INFO
==========
Brevi informazioni sull'utilizzo della web UI.

## DOWNLOAD

[ Cancella completati ]
Cancella le barre di progresso di tutti i download completati.

[ Info > Gestisci ]
Apre la webui nel path dove è stato scaricato il file.

[ Info > Riproduci ]
Riproduce il file scaricato (se audio/video).

[ Info > Agiungi alla playlist ]
Aggiunge il file scaricato alla playlist (se audio/video).

[ Info > Ferma ]
Ferma il download. Dopo alcuni secondi il download riparte automaticamente dall'inizio.

[ Info > Cancella ]
Elimina il file scaricato.


## GESTISCI

[ Agisci nel path ]
Seleziona il path di lavoro.

[ Spazio libero ]
Visualizza lo spazio libero nel path di lavoro.

[ Aggiungi link ]
Aggiunge il link del file da scaricare (il link verrà elaborato per avviare il download).

[ Edita link (coda) ]
Edita, aggiunge o elimina link nella coda (uno per riga).

[ Aggiungi XDCC ]
Aggiunge un comando xdcc per scaricare da server irc (il comando verrà elaborato per avviare il download).

[ Aggiungi torrent ]
Seleziona e aggiunge un file torrent da scaricare (il torrent verrà elaborato per avviare il download).

[ Leggi links.txt ]
Leggi o cancella il file links.txt (la lista di tutti i link processati).

[ Max download paralleli ]
Imposta il numero massimo di download paralleli nel path di lavoro.

[ Downloader ]
Imposta il downloader da utilizzare nel path di lavoro.

[ Reconnessione modem ]
Riconnette il modem/router. Per fare questo devi aggiungere il comando di riconnessione in Configurazione/Riconnessione modem. Da qui puoi anche visualizzare il tuo indirizzo IP.

[ Path status ]
Avvia o ferma il core di ZDL. Puoi anche terminare tutti i download attivi nel path di lavoro.


## CERCA

[ Cerca ]
Cerca file su server irc via xdcc.eu utilizzando uno o più termini di ricerca. Puoi selezionare il numero massimo di risultati.

[ Filtra ]
Rifinisce la ricerca filtrando i risultati con un termine.

[ Invia ]
Aggiunge il comando xdcc per scaricare il file (il comando verrà elaborato per avviare il download).


## PLAYLIST

[ Seleziona ]
Seleziona e aggiungi un file alla playlist.

[ Riproduci tutti i file audio ]
Riproduce tutti gli audio presenti nella playlist. Questa caratteristica può essere usata solo configurando cvlc, mpv o mplayer come media player in Configurazione/Media player.
Comandi:
 spacebar - Alterna pausa/play
 return - Traccia successiva
 ctrl+c - Esci/Chiudi

[ Scegli formato audio ]
Sceglie il formato per estrarre l'audio (mp3/flac).

[ Riproduci ]
Riproduce il file audio/video.

[ Rimuovi ]
Rimuove il file dalla playlist.

[ Audio ]
Estrae la traccia audio (mp3/flac) dal video.


## SOCKET

[ Nuovo socket ]
Avvia un nuovo socket sulla porta selezionata.

[ Vai al socket ]
Apre la webui sui socket avviati.

[ Termina socket ]
Termina un socket.

[ Termina socket attivo ]
Termina il socket corrente.

[ Termina tutti i download ]
Termina tutti i download attivi.


## CONFIGURAZIONE

[ Web UI utente ]
Imposta e cambia webui.

[ Lingua ]
Imposta e cambia la lingua.

[ Downloader predefinito ]
Imposta il downloader predefinito.

[ Connessioni Axel ]
Imposta il numero di connessioni per il downloader Axel.

[ Connessioni Aria2 ]
Imposta il numero massimo di connessioni per il downloader Aria2.

[ Max download paralleli ]
Imposta il numero massimo di download paralleli.

[ Sfondo xterm ]
Imposta il colore di sfondo per xterm.

[ Media player predefinito ]
Inserisce o seleziona l'applicazione per i file audio/video.

[ Text editor predefinito ]
Inserisce o seleziona l'applicazione per i file testo.

[ Browser predefinito ]
Inserisce o seleziona l'applicazione per accedere alla webui.

[ Aggiorna automatico ]
Abilita o disabilita gli aggiornamenti automatici di ZDL.

[ Sovrascrittura file ]
Abilita o disabilita la sovrascrittura dei file (riprendere o no un download).

[ Modalità avvio ]
Imposta come avviare ZDL.

[ Path nel lanciatore ]
Imposta il path in cui verrà avviato ZDL utlizzando il lanciatore nel menu.

[ Porta TCP torrent ]
Imposta la porta TCP predefinita per scaricare i torrent.

[ Porta UDP torrent ]
Imposta la porta UDP predefinita per scaricare i torrent.

[ Porta TCP socket ]
Imposta la porta predefinita per i socket.

[ Riconnessione modem ]
Seleziona il comando per riavviare il modem/router.

[ Account Web UI ]
Resetta l'account webui.


## CONSOLE

[ Registra solo gli errori ]
Registra solo gli errori nella console.

[ Leggi zdl_log.txt ]
Leggi o cancella zdl_log.txt (file contenente alcune informazioni sui download non riusciti).

[ Pulisci ]
Rimuove tutti gli eventi registrati nella console.


## ESCI

[ Esci ]
Chiude tutto e ferma il server.
