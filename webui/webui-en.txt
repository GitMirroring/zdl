WEBUI INFO
==========
Brief informations about the web UI usage.

## DOWNLOADS

[ Clean completed ]
Clean the progress bar of each completed download.

[ Info > Manage ]
Open the web ui on path where the file has been downloaded.

[ Info > Play ]
Play the downloaded file.

[ Info > Add to playlist ]
Add the downloaded file to the playlist.

[ Info > Stop ]
Stop the download. After few seconds the download automatically restart from beginning.

[ Info > Delete ]
Delete the file downloaded.


## MANAGE

[ Action path ]
Select the path to working on.

[ Free space ]
Display the available space in the action path.

[ Add link ]
Add the link of file to download (the link will be processed to start the download).

[ Edit links (queue) ]
Edit, add or delete links in the queue (one per line).

[ Add XDCC ]
Add an xdcc command to download from an irc server (the command will be processed to start the download).

[ Add torrent ]
Select and add a file torrent to download (the torrent will be processed to start the download).

[ Read links.txt ]
Read or delete the file links.txt (the list of all links processed).

[ Max parallel downloads ]
Set the max number of parallel download in the action path.

[ Downloader ]
Set the downloader to use in the action path.

[ Modem reconnect ]
Reconnect the modem/router. To do this you need to enter the reconnection command path in the Configuration/Modem reconnection. From here you can also display your IP address.

[ Path status ]
Start or stop the ZDL core. You can also kill all active downloads in the action path.


## SEARCH

[ Search ]
Search for a file containing specific term(s) on irc servers (via xdcc.eu). You can set the number of items to display in the table.

[ Filter ]
Refine the found items by filtering with a term.

[ Send ]
Add the xdcc command to download the file (the command will be processed to start the download).


## PLAYLIST

[ Select ]
Select and add a file to the playlist.

[ Play all audio files ]
Play all audio in the playlist. This feature can only be used by setting cvlc, mpv o mplayer as default media player in Configuration/Media player.
Commands:
 spacebar - Toggles pause/play
 return - Next track
 ctrl+c - Escape/Quit

[ Choose audio format ]
Choose the format to extract audio (mp3/flac).

[ Play ]
Play the audio/video file.

[ Remove ]
Remove file from playlist.

[ Audio ]
Extract audio (mp3/flac) from video.


## SOCKETS

[ New socket ]
Start a new socket on selected port.

[ Go to socket ]
Open the web ui on started sockets.

[ Terminate socket ]
Terminate a started socket.

[ Terminate active socket ]
Terminate the active socket.

[ Terminate all downloads ]
Terminate all active downloads.


## CONFIGURATION

[ User web UI ]
Set and change the web UI.

[ Language ]
Set and change the language.

[ Default downloader ]
Set the default downloader.

[ Axel connections ]
Set number of connection for Axel downloader.

[ Aria2 connections ]
Set the max number of connections for Aria2 downloader.

[ Max parallel download ]
Set the max number of parallel downloads.

[ Xterm background ]
Set the background color for xterm.

[ Default media player ]
Enter or select the application to play audio/video files.

[ Default text editor ]
Enter or select the application to edit text files.

[ Default browser ]
Enter or select the application to access the web ui.

[ Automatic update ]
Enable or disable automatic updates for ZDL.

[ Overwrite files ]
Enable or disable files overwriting (to resume or not a download).

[ Start mode ]
Set how to start ZDL.

[ Path in launcher ]
Set the path where ZDL will be started using the menu launcher.

[ Torrent TCP port ]
Set the default TCP port to download torrents.

[ Torrent UDP port ]
Set the default UDP port to download torrents.

[ Socket TCP port ]
Set the default port for sockets.

[ Modem reconnection ]
Select the command to reconnect the modem/router.

[ Web UI account ]
Reset your web ui account.


## CONSOLE

[ Records only errors ]
Record only errors in the console.

[ Read zdl_log.txt ]
Read or delete zdl_log.txt (a file containing some information about downloads failures).

[ Clean ]
Remove all recorded events in the console.


## EXIT

[ Exit ]
Close all and shutdown the server.
