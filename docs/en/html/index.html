ZigzagDownLoader (ZDL)

ZigzagDownLoader is a download manager:

  • equipped with command line user interfaces (CLI), for terminals (TUI), graphics (GUI) and for internet and web applications (Web-UI);
  • handles downloads of some downloaders such as Aria2, Axel, Wget, RTMPDump or cURL, using its potential (for example, download acceleration);
  • extracts the URL of the files to be downloaded from file hosting and video streaming services, also using youtube-dl extensions (or extractors);
  • downloads TORRENT;
  • automatically requests and downloads files via IRC/XDCC, without any human interaction via chat;
  • downloads movies LIVE from some television networks
  • extends web browsers, such as Conkeror, XXXTerm/Xombrero and, using Flashgot, Firefox;
  • the External Application Button extension integrates ZDL into Firefox, Chrome/Chromium and Opera;
  • using Cygwin, it also works in Microsoft Windows (CLI and TUI interfaces, also from Firefox + Flashgot of Windows).

The server and web interfaces, for remote and network control, make it a useful tool for a Network Attached Storage (NAS). The interfaces are responsive and suitable for different devices. In particular, they can also be used as a remote control for launching multimedia files already downloaded or being downloaded, using a web browser on mobile devices.

Furthermore, the Web interfaces (Web-UI) and graphics (GUI) allow the quick and intuitive search and selection of files via XDCC, without having to perform any IRC/chat. All interfaces allow live-tv recordings to be programmed.

But ZDL is also much more, because it can be:

  • combined with other programs or inserted into a script, as a work tool
  • extended by the user, who can develop new extensions suitable for their purposes
  • a polyglot system, using different interpreters (for PHP, Javascript, Python, Perl, Awk ...) and Bash tools

ZigzagDownLoader is a "zigzag" program: it is not just a program, it is a laboratory and a system open to new inventions.
