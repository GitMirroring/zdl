Torrent

Aria2 allows the download of torrents. This downloader does not have to be explicitly indicated to ZDL, but it must be installed correctly. A torrent can be started by simply sending the URL magnet (through all the systems provided for URLs in general, ZDL recognizes the magnet automatically) or by sending the file torrent, using the --torrent-file= option or the file marked with the .torrent extension.

The TCP and UDP ports can be set in the ZDL configuration (with the command zdl --configure or in the file $ HOME/.zdl/zdl.conf), explicitly indicated with the options --tcp-port= and --udp-port= or indicated in the Aria2 configuration. It is important that the ports are opened by the router.
